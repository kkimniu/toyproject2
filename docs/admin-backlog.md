## Remaining Work

### Issue Candidate 1

#### Title

관리자 작업 로그 대상 상세 진입 추가

#### Why

작업 로그에서 대상 ID만 보이면 실제 회원이나 신고 내용을 다시 찾는 데 시간이 든다.

#### Scope

- 회원 대상 로그에서 해당 회원 정보로 바로 이동
- 신고 대상 로그에서 해당 신고 정보로 바로 이동
- 대상이 삭제되었거나 접근할 수 없을 때의 표시 처리

#### Acceptance Criteria

- 관리자는 작업 로그에서 대상 회원 또는 신고로 바로 이동할 수 있다.
- 접근할 수 없는 대상은 깨진 링크 대신 명확한 상태로 표시된다.
- 로그 조회 흐름을 벗어나지 않고 다시 목록으로 돌아올 수 있다.
